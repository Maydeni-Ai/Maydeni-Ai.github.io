Comment ça marche ?

1 Achat de licence (one-time)

Le client achète une licence unique correspondant à son forfait. C'est un paiement une seule fois qui active son compte.

2 Phase de calibrage

L'IA calibre les données du client (prospects, produits, secteurs). Cette phase est 100% gratuite — aucune facturation ne démarre.

3 1 mois offert + abonnement

Une fois le calibrage terminé, le client bénéficie d'1 mois gratuit. Ensuite, l'abonnement mensuel démarre automatiquement.

Simulation interactive de l'application

🚀 Comment ça marche

1
Le directeur active la licence
Seul le directeur reçoit la clé de licence. Une seule activation suffit : la société, le dashboard et tous les accès sont configurés automatiquement.
2
Le directeur configure son équipe
Depuis le dashboard, il crée les comptes de ses agents commerciaux et leur communique leurs identifiants.
3
Les agents téléchargent l'app et se connectent
Les agents n'ont rien à configurer. Ils téléchargent l'application, entrent leur identifiant et mot de passe, et c'est parti. Aucune clé de licence requise côté agent.
✓
Tout est opérationnel !
GPS, visites, commandes, rapports — tout fonctionne immédiatement pour toute l'équipe.
